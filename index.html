Card Counting Website

This is a card counting practice website.

Counting cards is easy. Using the Hi-Lo method we can easily count cards in a game of blackjack by assigning values to cards and keeping a running count. In Hi-Lo we assign +1 to low cards (2-6), we assign 0 to middle cards (7-9), and we assign -1 to high cards (10-Ace).

The running count helps us determine the likelihood of favorable cards remaining in the deck. if the running count is positive the deck is loaded with high cards, which is advantageous for the player. Conversely if the running count is low then the deck is loaded with low cards, which is advantageous for the dealer.
